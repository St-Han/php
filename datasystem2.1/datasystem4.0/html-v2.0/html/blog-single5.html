类别影响力

影响力

  从气泡的分布来看，较大的气泡多集中在20世纪末到21世纪初，这表明在数据经济领域，某些早期的开创性研究具有很高的引用率和影响力。 近年来发表的文章虽然数量很多，但整体上影响力还需要时间来积累，因此较大的气泡数量相对较少。但是可以看到，这个领域直到现在都非常繁荣，时不时会出现一些引用量爆发性增长的文章。

趋势分析

  蓝色折线图显示了数据治理领域中每年文章发表数量的变化趋势。从图表中可以看到，自20世纪90年代末期开始，数据治理领域的研究逐渐增多，文章数量逐渐增加。2000年以后，文章发表数量呈现出显著增长，特别是在2010年至2015年之间，文章数量急剧增加，达到了峰值。但是在之后的几年里，文章数量出现了明显的下降。这可能反映了该领域某些研究热点的消退或研究方向的转变。 LSTM 模型预测未来几年文章数量将逐步恢复增长，尽管增速相对较慢。这表明数据治理领域的研究活动将在未来几年中有所恢复和增长，但不会像之前那样急剧上升。 预测的结果表明，虽然该领域在过去几年中经历了显著的波动，但未来几年可能会恢复增长，这反映了新的研究热点或技术发展可能会重新点燃学术界对该领域的兴趣。
